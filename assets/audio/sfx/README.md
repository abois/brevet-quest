# Effets sonores (SFX)

Fichiers attendus (chemin exact, format MP3) :

- `tap.mp3` — clic court (~150-300 ms)
- `correct.mp3` — bonne réponse
- `wrong.mp3` — mauvaise réponse
- `levelup.mp3` — montée de niveau
- `badge.mp3` — badge ou quête débloquée
- `swipe.mp3` — swipe vrai/faux

Sources gratuites suggérées : mixkit.co, freesound.org
